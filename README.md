## Featured Projects

### Gmail Inbox Agent

A polished dark Streamlit dashboard for reviewing and organizing Gmail inbox messages faster.

**Repository:** [Gmail_Inbox_Agent1](https://github.com/aryanbaki/Gmail_Inbox_Agent1)

**Features:**

- Demo Mode with 200 fake emails for safe public demos
- Optional Gmail Mode with local OAuth
- Fetches recent Gmail inbox messages
- Clusters similar emails using TF-IDF and KMeans
- Adds high, medium, and low priority labels
- Explains each priority decision
- Supports search, filters, charts, exports, and safe Gmail actions
- Keeps OAuth files private with `.gitignore`

**Tech used:** Python, Streamlit, Gmail API, OAuth, Pandas, scikit-learn, Plotly

### Visual Memory Search

A Streamlit app that lets users upload screenshots, analyze them with AI, and search through them using natural language.

**Repository:** [VisualMemorySearch](https://github.com/aryanbaki/VisualMemorySearch)

**Features:**

- Upload screenshots
- Analyze screenshot text and visual UI
- Store searchable screenshot metadata
- Create embeddings from screenshot summaries
- Search screenshots using natural language
- Return top matches with previews, confidence scores, and explanations
- Includes Demo Mode so the app can run without spending API credits

**Tech used:** Python, Streamlit, OpenAI vision and embeddings, Pandas, NumPy, scikit-learn, Pillow
